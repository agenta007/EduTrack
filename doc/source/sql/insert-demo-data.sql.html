Summary
This script populates the EduTrack database with initial demo data for testing and demonstration purposes. It begins by truncating all existing tables to ensure a clean state before inserting new records. The script defines five user roles including admin, headmaster, teacher, student, and parent. It then creates twenty-six user accounts with BCrypt hashed passwords for various staff and student personas across multiple schools. Finally, it establishes the foundational entities such as schools, classes, and schedules to support the school e-journal system. This setup allows developers to quickly verify system functionality without manual data entry.
