Summary
This component renders a dedicated page for viewing student absence records within the EduTrack system. It dynamically fetches data from the backend API, adjusting the request URL based on whether the current user is a student or an administrator. The interface displays a tabbed view that allows users to filter absence logs by specific academic terms. Key metrics such as total absences, excused count, and unexcused count are summarized using visual chips above the main data table. The table itself lists detailed information for each absence, including the subject, teacher, date, and approval status.
