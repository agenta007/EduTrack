Summary
This component manages the display and editing of student grades for a specific class within the EduTrack system. It fetches class details, student lists, schedules, and existing grades on mount, then organizes them into a tabular view filtered by academic term. The interface allows authorized users to add new grades via a dialog, automatically resolving schedule IDs and validating inputs. It dynamically calculates required grade counts based on weekly teaching hours and applies color-coded chips to indicate performance levels. The component handles role-based access control to restrict editing privileges to administrators, headmasters, and teachers.
