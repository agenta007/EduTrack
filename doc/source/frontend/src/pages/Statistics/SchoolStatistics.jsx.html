Summary
This component renders the school-level statistics dashboard for the EduTrack system by fetching and aggregating data for a specific school. It retrieves information about classes, teachers, and students, then calculates aggregate metrics such as average grades and absence rates across all classes. The interface allows users to filter these statistics by academic term using a tabbed navigation system. Visualizations include summary cards for key metrics and proportional bars that display grade distributions and per-class comparisons. Helper functions handle the logic for color-coding grade chips and formatting percentage values for the UI.
