Summary
This component serves as the entry point for recording student absences by allowing users to select a specific class. It fetches the list of available classes from the backend API and groups them by school name upon mounting. The interface displays these classes as clickable cards, each showing the class name and school year. When a user selects a card, the application navigates to a detailed absence recording page for that specific class. This view handles loading states and error messages to ensure a smooth user experience during data retrieval.
