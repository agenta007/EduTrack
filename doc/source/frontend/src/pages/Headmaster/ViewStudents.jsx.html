Summary
This component serves as the primary interface for headmasters to manage student enrollment within a specific school. It fetches current student lists and parent associations on mount, displaying them in a structured table with avatar and contact details. The page enables administrators to enroll new students via a dedicated dialog that queries available users and updates the local state upon success. It also allows for the expulsion of existing students through a confirmation dialog that removes the user from the system. Error handling and loading states are managed throughout to ensure a smooth user experience during data operations.
