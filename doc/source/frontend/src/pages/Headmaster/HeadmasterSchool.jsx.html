Summary
This component serves as the central management interface for a school's administrative settings within the EduTrack system. It allows headmasters to view and edit core school information such as name, address, and student capacity limits. The file also handles the configuration of academic term dates and the creation of a detailed daily schedule with various entry types like lectures and breaks. State management is used to track loading states, form inputs, and API response feedback for all these distinct sections. By consolidating these updates, it ensures that the school's operational data remains synchronized with the backend service.
