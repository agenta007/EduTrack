Summary
This component manages the parent-student relationship interface for the headmaster within the EduTrack system. It fetches and displays a list of parents linked to the current school, allowing the headmaster to link new parents to students, edit parent details, or unlink existing associations. The page handles asynchronous data retrieval and updates via Axios API calls, managing local state for loading indicators and error handling. It utilizes Material UI components to present a table of parents and modal dialogs for interaction. This file serves as the primary administrative view for managing family connections in the school database.
