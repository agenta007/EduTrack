Summary
This component renders the school information and daily schedule for the teacher's view within the EduTrack system. It fetches the current user's school details and associated schedule entries from the backend API upon initial load. The interface displays school metadata such as name, address, type, and headmaster name using Material UI typography components. It also presents the daily schedule in a table format, utilizing chips to visually distinguish between lectures, breaks, and special events. Error states and loading indicators are handled to ensure a smooth user experience during data retrieval.
