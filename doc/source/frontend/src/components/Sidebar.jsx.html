Summary
This component renders the application's navigation sidebar, dynamically adjusting its layout and visibility for both desktop and mobile views. It utilizes role-based logic to display specific menu items for administrators, headmasters, teachers, and parents, ensuring users only see relevant sections of the EduTrack system. The sidebar manages its own open and closed states while integrating with global contexts for authentication, student selection, and translation support. It handles nested submenus and responsive interactions, such as toggling menus on touch devices versus hover effects on desktop. By linking to various routes like grades, schedules, and profiles, it serves as the primary interface for navigating between different modules of the school e-journal.
