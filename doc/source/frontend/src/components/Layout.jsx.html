Summary
This component serves as the primary structural wrapper for the EduTrack application, establishing a consistent layout for all pages. It combines a collapsible sidebar with a main content area that dynamically adjusts its width based on the sidebar's state. The layout uses MUI's theme and transition utilities to ensure smooth animations when the sidebar opens or closes on different screen sizes. By wrapping child components in a SidebarProvider, it centralizes the sidebar state management across the entire application. This ensures that navigation remains accessible and consistent regardless of the specific page being viewed.
