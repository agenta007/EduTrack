Summary
This component renders a personalized dashboard header for the EduTrack system based on the current user's role. It displays a hero section with the user's avatar, name, email, and a color-coded role chip for immediate identification. The component dynamically generates a set of quick-access navigation tiles using a helper function that maps specific roles like admin, teacher, or student to relevant system paths. Each tile includes an icon and label that trigger client-side routing to the appropriate section when clicked. This ensures that users only see actions and links relevant to their permissions and current context within the school e-journal.
