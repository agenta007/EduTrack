Summary
This file provides global authentication state management for the EduTrack system using React Context. It defines the AuthProvider component which exposes login, logout, and token refresh capabilities to the entire application tree. The login function authenticates users via the backend and updates the global state with the JWT token and decoded user profile. A background timer automatically refreshes the access token every fifteen minutes to maintain session continuity. Additionally, an Axios interceptor ensures every outgoing API request includes the current Bearer token for authorization. A temporary mock login function is included to facilitate rapid development and testing without real credentials.
