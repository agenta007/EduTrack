Summary
This component acts as a security guard for specific routes within the EduTrack application by verifying user authentication and authorization before rendering content. It utilizes a custom useAuth hook to retrieve the current user's data and checks if the authentication process is still in progress. If the system is loading, it displays a circular progress indicator to provide visual feedback to the user. Once loading completes, it checks if the user is logged in and redirects them to the login page if they are not. If the user is logged in but lacks the required role specified in the props, they are redirected to an unauthorized page. Otherwise, it renders the protected child components, ensuring only authorized personnel can access sensitive parts of the school e-journal system.
