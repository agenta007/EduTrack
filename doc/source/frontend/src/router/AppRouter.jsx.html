Summary
This file configures the client-side routing for the EduTrack application using React Router. It defines URL paths for all major user roles, including administrators, headmasters, teachers, parents, and students. Each route is wrapped in a ProtectedRoute component to enforce role-based access control before rendering the corresponding page component. The system also handles public access for login and unauthorized states, while redirecting unauthenticated users to the appropriate landing page. This centralizes navigation logic and ensures secure, role-specific content delivery throughout the school e-journal system.
