Summary
This controller manages school-related operations within the EduTrack system by exposing REST endpoints under the /api/schools path. It handles core CRUD activities for schools, including listing, retrieving, creating, updating, and deleting school records. The controller enforces strict role-based access control, restricting most administrative actions to users with the ADMIN role while allowing headmasters limited update capabilities. It delegates business logic to the SchoolService layer and utilizes DTOs for data transfer. Additionally, it supports specific operations for updating school information and managing associated profiles and schedules. This component serves as the primary interface for external clients to interact with school data in the e-journal application.
