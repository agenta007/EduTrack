Summary
The UserService class manages user lifecycle operations and data retrieval for the EduTrack school e-journal system. It handles creating, updating, and fetching user profiles while ensuring email uniqueness and secure password encoding. The service provides specialized methods to retrieve teachers, students, parents, and headmasters filtered by school or role. It also supports profile picture retrieval by returning image bytes with appropriate content types. This service acts as the central business logic layer for user-related interactions within the application.
