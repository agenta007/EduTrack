Summary
ProfileService manages user profile operations within the EduTrack system by handling authentication context through Spring Security principals. It provides methods to retrieve user details, securely update passwords with validation, and manage profile picture uploads or deletions. The service also allows users to update their biography with length constraints to ensure data integrity. Each operation validates input and persists changes to the database via the UserRepository. This service acts as a central point for modifying user account metadata and credentials. It ensures that all profile updates are tied to the currently authenticated user.
