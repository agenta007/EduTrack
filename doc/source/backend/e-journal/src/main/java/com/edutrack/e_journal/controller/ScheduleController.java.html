Summary
This controller manages curriculum scheduling endpoints for the EduTrack system, allowing users to view and modify which teachers teach which subjects to specific classes. It exposes RESTful operations to list schedules by class or teacher, create new assignments, update lecture types, and delete entries. Access control is strictly enforced, restricting creation and modification actions to administrators and headmasters while allowing broader read access for teachers, parents, and students. The implementation relies on a dedicated service layer to handle business logic and data persistence. Swagger annotations are included to document the API for developers.
