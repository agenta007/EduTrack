Summary
This file defines the Teacher entity which extends the base User model to represent educators within the EduTrack system. It uses a one-to-one relationship with the User table to share the primary key while adding specific teacher-related data. The entity links each teacher to a specific school and stores their monthly gross salary in Bulgarian Lev. Additionally, it maintains a many-to-many relationship with the Subject table to track the teacher's professional qualifications. This structure allows the system to manage teacher profiles, assignments, and compensation efficiently.
