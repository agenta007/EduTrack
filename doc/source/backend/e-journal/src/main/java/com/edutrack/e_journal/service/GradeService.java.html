Summary
This service manages the lifecycle of academic grades within the EduTrack system, handling retrieval, creation, and deletion operations. It enforces strict validation by ensuring all new grades conform to a specific Bulgarian grading scale before persisting them to the database. The implementation supports fetching grades by class, current authenticated student, or specific student ID, mapping complex entity relationships into simplified DTOs for API responses. It also securely resolves user identities from Spring Security principals to associate grades with the correct students and teachers. By leveraging Spring Data repositories, it efficiently queries and updates grade records while maintaining data integrity through existence checks.
