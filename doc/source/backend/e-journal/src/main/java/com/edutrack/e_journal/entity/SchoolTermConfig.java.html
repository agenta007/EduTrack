Summary
This entity defines per-school configuration for term and school-year date boundaries within the EduTrack system. It maintains a one-to-one relationship with the School entity, sharing the same primary key to ensure each school has exactly one configuration record. The class stores key dates as MM-dd strings, including the start of the school year and the start of the second term. It also specifies distinct end dates for different grade levels, such as elementary, progymnasium, and gymnasium. This configuration allows the system to dynamically calculate academic periods based on individual school policies.
