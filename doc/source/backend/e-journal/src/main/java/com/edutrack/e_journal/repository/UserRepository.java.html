Summary
This interface defines the data access layer for user management within the EduTrack system by extending Spring Data's CrudRepository. It provides standard methods to locate users by email and verify email uniqueness to prevent duplicate accounts. The repository also supports role-based queries to retrieve all users assigned to a specific security role. Two custom JPQL queries identify available students and teachers who have not yet been enrolled or assigned to a school. This functionality is critical for the onboarding process, allowing the system to efficiently find unassigned users for new school enrollments.
