Summary
This configuration class establishes the security framework for the EduTrack school e-journal system by integrating Spring Security with JWT-based authentication. It defines a stateless session policy and disables CSRF protection to support a modern API architecture. The setup exposes essential beans for password encoding and authentication management while configuring a custom filter chain that validates requests via a JWT filter and a local admin filter. Public endpoints for authentication and Swagger documentation are explicitly permitted, whereas all other requests require authentication. Additionally, it configures CORS to allow cross-origin requests from the frontend development server.
