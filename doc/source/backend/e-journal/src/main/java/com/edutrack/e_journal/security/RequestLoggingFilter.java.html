Summary
This component implements a servlet filter that logs details for every incoming HTTP request in the EduTrack system. It extends OncePerRequestFilter to ensure exactly one execution per request and uses SLF4J to record the HTTP method, URI, response status, processing time, and client IP address. The filter captures the start time before delegating to the next filter in the chain and logs the duration after the response is generated. It also handles proxy headers to accurately identify the original client IP address. As a high-priority component, it provides essential visibility into request flow and performance for debugging and monitoring purposes.
