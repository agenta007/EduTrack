Summary
This controller manages headmaster operations for hiring, firing, and managing teacher salaries within the EduTrack system. It exposes REST endpoints to list teachers by school, view available unassigned teachers, and perform administrative actions like creating new teacher accounts or updating compensation. All endpoints are strictly protected, requiring either ADMIN or HEADMASTER roles to ensure proper authorization. The class delegates business logic to the TeacherService, handling complex tasks such as verifying school ownership and processing user creation. It also integrates Swagger annotations to document the API for external consumers. This component serves as the primary interface for school leadership to oversee staff composition and payroll.
