Summary
This controller manages HTTP endpoints for querying school classes within the EduTrack system. It exposes REST APIs to retrieve all classes, filter classes by school, fetch specific class details, and list enrolled students. The implementation relies on the ClassService for business logic and uses Spring Security annotations to enforce role-based access control. Different endpoints are restricted to specific user roles such as ADMIN, TEACHER, or PARENT to ensure data privacy. Swagger annotations are included to document the API operations for developers. This component serves as the primary interface for class-related data retrieval in the application.
