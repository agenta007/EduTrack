Summary
This controller manages disciplinary complaint records within the EduTrack system by exposing RESTful endpoints under the /api/complaints path. It handles operations for listing complaints by class or specific student, allowing authorized users like teachers and administrators to view records. The system also permits students to retrieve their own complaint history and enables authorized staff to file new complaints or delete existing ones. Access to these endpoints is strictly controlled via role-based authorization to ensure data security. The implementation relies on a dedicated service layer to process business logic and returns standardized DTOs to the client.
