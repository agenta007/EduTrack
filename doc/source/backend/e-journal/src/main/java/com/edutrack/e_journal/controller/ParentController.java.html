Summary
This controller manages administrative operations for parent accounts within the EduTrack system. It exposes REST endpoints that allow headmasters and administrators to list parents, link or unlink them to students, and update parent details. All endpoints require authentication and enforce role-based access control to ensure only authorized personnel can modify school data. The controller delegates business logic to the ParentService layer while handling HTTP request mapping and response formatting. It serves as the primary interface for managing the relationship between parent users and student records in the e-journal application.
