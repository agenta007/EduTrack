Summary
This class defines the data structure for submitting a new grade entry within the EduTrack system. It utilizes Lombok annotations to automatically generate getters and a no-argument constructor, ensuring clean and concise code. The model enforces strict validation rules to guarantee data integrity during the submission process. Specifically, it requires non-null identifiers for both the student and the specific class schedule. The grade value is constrained to a BigDecimal type, restricted to a range between 2.00 and 6.00 to match the institution's grading scale. This DTO serves as the primary input contract for the backend service handling grade updates.
