Summary
The AbsenceService manages the lifecycle of student absence records within the EduTrack system. It provides methods to retrieve absences by class, authenticated student, or specific student ID, converting database entities into DTOs for API responses. The service handles creating new absence entries by validating associated student and schedule data before persistence. It also supports toggling the excused status of existing records and deleting them by ID. A helper method resolves authenticated users from security principals to ensure proper access control. This service acts as the core business logic layer for attendance tracking operations.
