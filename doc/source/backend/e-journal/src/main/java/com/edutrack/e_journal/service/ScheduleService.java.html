Summary
The ScheduleService manages the lifecycle of class schedules within the EduTrack system by handling creation, retrieval, updating, and deletion operations. It interacts with multiple repositories to validate associated classes, subjects, and teachers before persisting new schedule entries or modifying existing ones. The service provides specialized methods to fetch schedules for specific classes, individual teachers, or the currently authenticated user. It also supports partial updates to lecture types and attendance tracking settings without requiring full entity replacement. Internal helper methods map database entities to flattened DTOs to simplify frontend consumption and avoid deep object nesting. This service acts as the core business logic layer for all schedule-related data processing in the application.
