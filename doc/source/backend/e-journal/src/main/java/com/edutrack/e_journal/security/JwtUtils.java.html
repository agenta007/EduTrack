Summary
This component manages JSON Web Token operations for the EduTrack system by generating and validating access and refresh tokens. It creates access tokens containing user details like ID, email, and role, while refresh tokens are shorter-lived and contain only the subject. The class uses a configured secret key to cryptographically sign these tokens to ensure their integrity and authenticity. It provides methods to extract the user email from a token and to validate whether a token is still active and properly signed. This utility ensures secure authentication flows by handling token lifecycle management within the application's security layer.
