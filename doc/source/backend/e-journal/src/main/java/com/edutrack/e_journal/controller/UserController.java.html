Summary
This controller manages user administration endpoints for the EduTrack system, handling CRUD operations and role-based queries. It exposes REST APIs for creating, updating, and deleting user accounts, as well as retrieving lists of users filtered by role such as teachers, students, parents, and headmasters. Access to these endpoints is strictly controlled, with most administrative actions restricted to users with the ADMIN role and certain queries also available to HEADMASTERs. The class delegates business logic to a UserService layer, ensuring separation of concerns while returning standardized DTOs to the client. It also supports retrieving user profile pictures as raw image bytes. This component serves as the primary interface for managing the system's user base and their associated school data.
