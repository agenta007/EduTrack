Summary
This configuration class sets up the OpenAPI documentation for the EduTrack school e-journal system using Swagger. It defines a global JWT Bearer authentication scheme named bearerAuth that applies to all endpoints except the three authentication endpoints. The OpenAPI bean provides metadata including the API title, version, and detailed descriptions of the five user roles and their permissions. It also includes contact information for the development team to assist with API inquiries. This setup ensures that developers and consumers of the API have clear, standardized documentation for integration and testing.
