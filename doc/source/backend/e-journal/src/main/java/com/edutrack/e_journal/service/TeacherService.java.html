Summary
The TeacherService class manages the lifecycle of teacher accounts and their association with schools within the EduTrack system. It provides methods for headmasters to hire new teachers, assign existing users to schools, and terminate employment by removing school links. The service enforces strict access controls to ensure only authorized headmasters can modify teacher records for their specific institutions. It handles data validation by checking for duplicate emails and preventing teachers from being assigned to multiple schools simultaneously. Additionally, it supports retrieving teacher lists and updating salary information while maintaining data integrity through repository interactions.
