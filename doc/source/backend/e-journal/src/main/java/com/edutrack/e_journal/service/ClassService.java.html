Summary
ClassService manages the retrieval and mapping of school class data for the EduTrack system. It provides methods to fetch all classes, classes by school, or a specific class by ID, ensuring proper authorization checks for administrators and headmasters. The service also retrieves student lists associated with a given class. It relies on Spring Data repositories to access SchoolClass, Student, and User entities from the database. Internal helper methods convert entity objects into Data Transfer Objects for API responses. This service acts as a central business logic layer for class-related operations within the application.
